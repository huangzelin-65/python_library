# 图书馆管理系统流程说明
以下是系统的整体主流程流程图：

flowchart TD
    A[程序启动] --> B[初始化LibrarySystem对象，添加初始图书和用户]
    B --> C[进入主循环，显示主菜单]
    C --> D{用户选择主菜单选项}

    %% 主菜单分支
    D -->|1. 图书管理| E[进入图书管理子菜单]
    D -->|2. 用户管理| F[进入用户管理子菜单]
    D -->|3. 借阅图书| G[执行借阅图书操作]
    D -->|4. 归还图书| H[执行归还图书操作]
    D -->|5. 退出系统| I[打印退出提示，结束程序]
    D -->|输入错误| J[提示输入错误，返回主菜单C]

    %% 图书管理子菜单分支（修正区域）
    E --> K{用户选择图书管理子选项}
    K -->|1. 添加图书| L[输入信息，调用add_book方法]
    K -->|2. 删除图书| M[输入编号，调用delete_book方法]
    K -->|3. 修改图书信息| N[输入编号，调用modify_book方法]
    K -->|4. 按书名搜索| O[输入关键词，调用search_book(title)方法]
    K -->|5. 按作者搜索| P[输入关键词，调用search_book(author)方法]
    K -->|6. 按ISBN搜索| Q[输入关键词，调用search_book(isbn)方法]
    K -->|7. 查看所有图书| R[调用show_all_books方法]
    K -->|8. 返回主菜单| C
    K -->|输入错误| S[提示输入错误，返回图书管理子菜单E]
    L --> E
    M --> E
    N --> E
    O --> E
    P --> E
    Q --> E
    R --> E

    %% 用户管理子菜单分支
    F --> T{用户选择用户管理子选项}
    T -->|1. 添加用户| U[输入信息，调用add_user方法]
    T -->|2. 查看所有用户| V[调用show_all_users方法]
    T -->|3. 返回主菜单| C
    T -->|输入错误| W[提示输入错误，返回用户管理子菜单F]
    U --> F
    V --> F

    G --> C
    H --> C
